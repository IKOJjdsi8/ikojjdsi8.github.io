Privacy Policy

This app is a personal project created to access and review authorized Pinterest account data for internal reporting, account insights, and educational API testing in Postman.

This app uses the Pinterest API and is not affiliated with, endorsed by, or sponsored by Pinterest.

What data may be accessed

  • Pinterest account basic information
  • OAuth authorization data required to make API requests
  • Other Pinterest account data explicitly authorized by the user

How data is used

  • Only with the user’s authorization
  • Only to provide services to that same user
  • For personal reporting, account insights, and educational API testing
  • Not sold, shared, or redistributed to third parties
  • Not combined with data from other accounts or services for resale or profiling

Data retention and deletion

If Pinterest is disconnected from this app, any stored Pinterest access tokens and Pinterest-derived data used by the app will be deleted.
